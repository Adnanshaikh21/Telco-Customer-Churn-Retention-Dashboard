# Telco Customer Churn Retention Dashboard


## 📋 Project Overview
Excel-based end-to-end analysis of telecom customer churn using a **7,043-customer dataset**. Performed data cleaning, feature engineering, and built interactive PivotTable-based dashboard with custom risk scoring logic based on contract type, services, tenure, and charges.

## 📊 Key Insights
- Total customers: **7,043**
- Churn rate: **26.5%** (1,869 churned customers)
- Active customers: 5,174
- Average monthly charge: **$64.76**
- Highest churn observed in Month-to-month contracts

## ✨ Features & Analysis Performed
- Custom churn risk scoring model (derived from contract, services, tenure, charges)
- Cohort-style analysis using tenure buckets
- Churn breakdown by demographics (gender, partner, dependents, senior citizen)
- What-If style recommendations for churn reduction strategies
- Interactive PivotTables and charts for filtering

## 🛠 Tools & Techniques
- **Microsoft Excel** (Advanced functions, PivotTables, PivotCharts, data cleaning)
- Feature engineering (tenure periods, monthly/tenure stayed/churned metrics)
